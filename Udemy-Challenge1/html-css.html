HTML and CSS Summary

HTML SUMMARY

HTML(Hyper Text Markup Language) is used to define our page content, structure and meaning. You don't use it for stlying purposes. Use CSS for the instead

  • HTML uses *elements* to describe content
  • HTML elementsy typically have an opening tag, comments and closing tag.
  • You can also have void (empty) elements like images
  • You can also configure elements with attributes
  • There's a long list of available elements but you gain experience over time.

Learn more about all available HTML elements on MDN HTML Element reference

CSS Summary

CSS (Cascasding Style sheets) is used for styling your page content

  • Styles are assigned via property value-pairs
  • You can assing styles via the "styles" attribute
  • To avoid code duplication, you typically use global styles instead
  • Alternatively you can work with external sytles sheet you can link to
  • When working with CSS concepts like inheritance, specificity and box model should be understood.

Learn more about all available CSS properties and values on the MDN CSS property references.
